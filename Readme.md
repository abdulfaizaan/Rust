## 💡 Key Takeaways & Rust Concepts
> 💡 **Tip:** Use this space to write down "Aha!" moments.
* **Immutability by Default:** In Rust, variables are immutable by default. You explicitly need `let mut x = 5;` to change it later.
* **The Borrow Checker:** Rust enforces memory safety at compile-time, meaning no garbage collector and no dangling pointers!
